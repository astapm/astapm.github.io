KXStitch: создание схем вышивок крестом

KXStitch – программа для создания шаблонов вышивок крестиком, как с нуля, так и при помощи изображений. Лицензия: . ОС: Linux, UNIX. Лицензия: GNU GPL. Сайт: kxstitch.sourceforge.net

Open Source программы существуют даже для любителей вышивания, которым хочется автоматизировать создание и редактирование шаблонов вышивки. Занятия вышиванием замечательно успокаивают нервную систему и могут рекомендоваться даже программистам. Ведь несомненно лучше в качестве отдыха от работы не играть в бесполезные игры, а вышить, например, на наволочке портрет Линуса Торвальдса.

Свободная программа в помощь вышивальщикам называется KXStitch, и она специально создана для систем GNU/Linux. Для её работы потребуется библиотеки Qt и KDE.

KXStitch это программа, которая позволяет создавать схема вышивки крестиком. Это древний вид рукоделия вышивания рисунков на канве с помощью иглы и цветных нитей.

Нельзя сказать, чтобы KXStitch была волшебной программой и мечтой вышивальщицы, которая позволяла бы только загрузить картинку, ткнуть кнопочку и тут же получить готовую схему с расшифровкой по палитре цветов нитей для разных производителей.

Тем не менее, программа позволяет создавать схемы вышивки не только с нуля на основании сетки. Создание схем для вышивки из изображений тоже возможно. В программе предлагаются инструменты для уменьшения количества цветов изображения и производства окончательного дизайна шаблона для вышивания. Также картинки можно использовать в качестве фона при создании схемы вышивки с нуля.

В набор предлагаемых инструментов входят: создание прямоугольников и эллипсов, дублирование областей, повороты выделенных частей схемы. Существует также библиотека шаблонов, которую можно использовать для хранения небольших моделей, и которые затем могут быть повторно использованы в других узорах. Узоры в этих библиотеке хранятся в виде иерархического списка, что позволяет сортировать и управлять ими.

Когда схема готова, её можно распечатать в нескольких форматах. Есть варианты распечатки с инструкциями и таблицей мулине, количеством используемой нити и количеством стежков.

Говоря о программах для вышивания, можно напомнить и специальное расширение Stitch-art для графического редактора GIMP, которое тоже предназначено для создания схем вышивок.
