FIGlet и Ascii Design: создание текста в виде ASCII графики

FIGlet и Ascii Design — это консольная программа и графический для неё фронтенд, которые предназначены для создания текстовых баннеров в виде ASCII графики. Лицензия: Open Source. Сайты: FIGlet, Ascii Design.

Создавать картинки и изображения из букв-символов — идея не новая. Ещё в былые времена, например, был распространён изящный жанр стихографики и каллиграммы, когда слова стихотворения образуют рисунок. Французский поэт Гийом Аполлинер был мастером каллиграммы.

В наш век компьютерных мощностей создание изображений из букв-знаков называется ASCII графикой, так как изображения создаются при помощи буквенных, цифровых символов и символов знаков пунктуации из числа 95 символов стандартной американской символьной таблицы ASCII, используемой для вывода информации в компьютерном терминале. Компьютер позволяет создавать эффектную ASCII графику, в том числе и анимированную, претендующую на отдельный жанр изобразительного творчества. Хотя при этом потеряно изящество каллиграмм прошлых столетий и текст в ASCII графике не имеет никакого самостоятельного смысла и служит исключительно палитрой для создания картинки.

ASCII графика может обладать некоторой практичностью, как, например, в случае программы FIGlet. Её название составлено из первых букв имён её авторов — Frank, Ian and Glen’s letters. Это старенькая программа со времён 90-х годов позволяет прямо из командной строки создавать текстовые баннеры с помощью ASCII графики — то есть создавать текст, где «текстовой шрифт» выполнен в стиле ASCII-картинок, и который можно открыть в простом текстовом редакторе. Причём ASCII-шрифтов для FIGlet сделано немало. Есть даже один вариант кириллицы moscow.

Из подобных текстов можно делать в некоторых случаях те же рекламные баннеры. Так же в конспирологических целях можно учитывать, что смысл подобных текстов не поддаётся анализу компьютерных ботов, — например, ботов АНБ, которые, например, следят за перепиской в электронной почте.

Программа FIGlet имеется практически для всех ОС: Linux, Amiga, Apple, MS-DOS, Windows. Работает программа в режиме командной строки, когда текст для преобразования в ASCII картинку передаётся в виде параметра для программы.

$ figlet Hello
 _   _      _ _         
| | | | ___| | | ___    
| |_| |/ _ \ | |/ _ \    
|  _  |  __/ | | (_) |   
|_| |_|\___|_|_|\___/

Также в программе есть интерактивный режим, который работает, если запустить программу без передачи ей текста для преобразования, без или только с аргументами. Основные аргументы FIGlet, которые можно задавать программе:

  • -f для выбора файла шрифта. Шрифты можно скачать со специального хранилища ftp.figlet.org. Сами шрифты — это обычные текстовые файлы с расширением flf, в которых уже нарисованы все буквы алфавита, и их можно посмотреть в любом текстовом редакторе, не используя специальных программ.
  • -d для смены директории со шрифтами.
  • -c расположить вывод по центру.
  • -l расположить вывод слева.
  • -r расположить вывод справа.
  • -t установить ширину вывода по ширине терминала.
  • -k кернинг, то есть символы будут стоять отдельно, не сливаясь друг с другом.

Существует и графический фронтенд к FIGlet, который называется Ascii Design, и который позволяет делать всё то же самое, что и FIGlet, но не из командной строки, а в удобном графическом окошке.

В поле Original text вводим нужный текст, а в поле ASCII Art text получаем готовый ASCII текстовой баннер.

Ascii Design доступна для систем Linux и Windows. Размер программы около 300 кбайт.
