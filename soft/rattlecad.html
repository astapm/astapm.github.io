RattleCAD: изобретаем велосипед

RattleCAD — специализированная программа для разработки новых моделей велосипедов, начиная с общего дизайна и заканчивая отдельными компонентами. Лицензия: Open Source. Сайт: rattlecad.sourceforge.net/

Существует такое обидное прозвище «изобретатель велосипедов». Обычно так называют человека, придумывающего очевидные вещи.

Тем не менее есть вполне уважаемая работа по созданию новых моделей настоящих велосипедов. Есть даже специальное ПО, рассчитанное только на разработку велосипедов. Примером открытой и бесплатной программы для велосипедостроительства без кавычек является rattleCAD.

RattleCAD — программа для тех, кому хочется создать свой собственный велосипед. RattleCAD проведет вас через процесс проектирования, начиная с создания базовой геометрии и дизайна до усовершенствования отдельных компонентов велосипедной рамы.

Соответственно всем шагам процесса создания велосипеда соответствуют закладки в главном окне программы. Это закладки: базовая концепция, параметры рамы, отдельных элементов и параметры труб рамы, макет. Параметры изменяются посредством отдельной специальной конфигурационной панели. Эта панель является контекстной, в которой параметры автоматически отображаются в зависимости от выбранной вкладки.

На основе этого процесса rattleCAD создает рабочие чертежи. Программа использует формат XML файла для хранения информации непосредственно по проекту и формат SVG файл для представления отдельных элементов макета велосипеда.
